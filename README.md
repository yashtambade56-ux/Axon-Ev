# Axon EV Car Comparison

**Final HTML Project:**  
A fully responsive web page designed to compare Axon EV cars. Users can view detailed specifications like range, battery capacity, charging time, and price in a clean, easy-to-read layout. This project demonstrates HTML, CSS, and basic web design skills.

---

##live demo
https://yashtambade56-ux.github.io/Axon-Ev/
---

## Project Overview
The Axon EV Car Comparison page allows users to:
- Quickly compare specifications of different EV models.
- Check important details such as range, battery, charging time, and price.
- Access a mobile-friendly and responsive layout that works on all devices.

This project is perfect for beginners learning web development or anyone wanting a **clean, functional car comparison page**.

---

## Features
- Clean single-page layout for car comparison
- Responsive design using CSS Grid and Flexbox
- Displays key car specifications:
  - Model Name
  - Range
  - Battery
  - Charging Time
  - Price
- Easy-to-read design suitable for desktop and mobile
- Optional: can be extended with JavaScript for interactive features

---

## Tech Stack
- **HTML5** – structure and content  
- **CSS3** – styling, Flexbox, Grid, and responsive layout  
- **JavaScript** (optional) – for future interactive features
